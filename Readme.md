# Тестовое для стажировки в VK

**Тестовое задание: Анализ IP-адресов**

Написать консольное приложение на С#

**Что нужно сделать:**   
- Загрузите все ip адреса из файла в память
- Создайте функцию, которая ходит в аpi https://ipinfo.io/{ip}/json (GET запрос) и возвращает объект lpData (с необходимыми полями из вернувшегося ответа)
- Определите и выведите, iр адреса каких стран представлены в файле, а также их количество на каждую страну (например, Ru - 5; Jp - 1; …).
- Возьмите за основу ту страну, у которой больше всего представленных ір адресов и выведите город каждого (например, Ru: Moscow, St. Petersburg, …)

**Ограничения**
- Разрешено использовать Newtonsoft.Json, другие внешние пакеты запрещены.
- Все необходимые значения необходимо вычислять кодом.

**Использованные технологии**
- .NET 8.0
- HttpClient для запросов
- Newtonsoft.Json для десериализации

**Как запустить**
1. Положите файл `IPs.txt` рядом с проектом
2. `dotnet restore`
3. `dotnet build`
4. `dotnet run`

**Пример вывода**
Найдено 11 IP-адресов. Начинаем обработку...
Страны и количество IP-адресов:
DE - 3
RU - 2
...
Страна с наибольшим количеством IP: DE (3 шт.)
DE: Düsseldorf, Falkenstein, Frankfurt am Main
